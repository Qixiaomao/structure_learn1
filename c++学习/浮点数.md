# 浮点数

```

题目描述：
链接：https://ac.nowcoder.com/acm/problem/21986
来源：牛客网

会说整数之后，牛牛开始尝试浮点数（小数）

输入一个浮点数，输出这个浮点数。
输入描述:
输入一个浮点数
输出描述:
输出一个浮点数，保留三位小数
示例1
输入
复制
1.359578
输出
复制
1.360

```

## c++中输出有效位数的方法

```
#include<iostream>
#include<iomanip>
using namespace std;

int main(){
    double d;
    cin >> d  ;
    cout << fixed << setprecision(3);
    cout << d ;
    return 0;
}
```

### 浮点数

```
链接：https://ac.nowcoder.com/acm/problem/21992
来源：牛客网

输入两个整数a, b, 输出a除以b的值，保留三位小数
输入描述:
输入两个整数，在int范围内
输出描述:
输出一个浮点数，保留三位小数
示例1
输入
复制
5 2
输出
复制
2.500
```

### 代码

```
#include<iostream>
#include<iomanip>
using namespace std;

int main(){
    double a;
    double b;
    double c;
    cin >> a;
    cin >> b;
    c = a/b;
    cout << fixed << setprecision(3) << c;
    return 0;
}

```
